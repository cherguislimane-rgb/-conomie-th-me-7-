# Thème 7 — Site (GitHub Pages)

## Mise en ligne rapide
1. Crée un dépôt GitHub (ex: `theme7`).
2. Dépose **le contenu** de ce dossier à la racine du dépôt.
3. Sur GitHub : **Settings → Pages → Build and deployment**
   - Source: `Deploy from a branch`
   - Branch: `main` (ou `master`), dossier `/ (root)`
4. GitHub te donne l'URL du site.

## Contenu
- `index.html` : page d'accueil (image cliquable + liste)
- `pages/` : une page par capacité
- `assets/` : images
- `styles.css` : styles communs
